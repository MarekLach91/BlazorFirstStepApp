Pierwsze wykorzystanie frameworka Blazor

5 stron i 5 komponentów

1 Formularz rejestracji z wykorzystaniem własnego komponentu z inputem i walidacja (użycie: EventCallback @bind)<br>
2. Pogoda tabela - rozbudowa domyślnych danych i wykorzystanie Virtualize<br>
3. Lista wydmuszkowych produktów na bazie komponentu - kafla produktu<br>
4. Historia transakcji - tabela z pobieraniem danych i filtrowaniem rekordów, komponent z komunikatem błędu, przekierowanie (IJSRuntime, Route Parameter)<br>
5. Rozbudowany formularz z danymi i walidacja. Wykorzystanie komponentu z dialog. (użycie EditForm z DataAnnotationsValidator, IJSRuntime)<br>
6. Globalny komponent i możliwość zmiany koloru tła głównej części. <br>

Przypisanie plików css do poszczególnych componentów i stron.
